Welcome to the Physical Oceanography Repository

I've set up a github repository with some useful setup files. I hope these are useful and straight forward. You can download the whole folder using the links at the top of the page or follow the links below to individual files. The full wiki and repository is here

Main features of the repo:

  • New to the unix systems? Take a look at example_.cshrc and example_.login .
  • Fed up of csh? Take a look at bash and example_.bashrc
  • Kick that MATLAB habit? This is how to set up Anaconda and Ipython notebooks
  • How am I putting this all up here? Git! Set it up for yourself here
  • Useful topics from NCAS software carpentry, on everything from UNIX to parallel programming and much more! Found here
  • Easy manipulation of netcdf NCO notes
