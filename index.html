UNIX:

    Most powerful systems run on Linux(UNIX) OS & many big models and public domain software packages assume a linux enviroment. Linux works through interpretation of commands by a shell in user defined environment.

  • New to the unix systems? Take a look at example_.cshrc and example_.login(These help define that environment to autmatically load in software etc)
  • Swapping shells? Take a look at bash and example_.bashrc
  • To get started: Top 30 Unix commands.

PYTHON:

    Python is an open source programming language with many "tool boxes" and a growing user base. It's well established on NOCs and has an increasing userbase in OES. For those thinking about switc hing to Python it's worth noting it's promotes better programming techniques and handles large datasets well. Personally the fact is open source and you can share your scripts with anyone regardless of whe ther they can afford expensive propriatory software is also pretty important! If you're worried about user friendlyness, fear no more! Python now has great GUIs such as Spyder and Ipython Notebooks!
